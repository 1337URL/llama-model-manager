Rules are applied in order to proxied responses based on matching conditions.

Rule Format: JSON array of rule objects. Each rule has a match section (URL pattern, content type, method) and a transform section (type and action).

Rule Examples

Remove sensitive fields from JSON

Example Rule:

{ "match": { "url": "https://api.example.com/v1/*", "content_type": "application/json" }, "transform": { "type": "json", "action": "remove", "fields": ["password", "secret_token"] } }

Removes specified fields from matching JSON responses.

Add fields to JSON responses

Example Rule:

{ "match": { "url": "https://api.example.com/*", "content_type": "application/json" }, "transform": { "type": "json", "action": "add", "fields": { "processed_by": "llama-manager", "timestamp": "2026-04-17T00:00:00Z" } } }

Adds new fields to matching JSON responses.

Rename JSON keys

Example Rule:

{ "match": { "url": "https://api.example.com/v1/users", "content_type": "application/json" }, "transform": { "type": "json", "action": "rename", "renames": { "id": "user_id", "name": "username" } } }

Renames specified keys in matching JSON responses.

Set default values for missing fields

Example Rule:

{ "match": { "url": "https://api.example.com/v1/*", "content_type": "application/json" }, "transform": { "type": "json", "action": "set", "fields": { "active": true, "verified": false } } }

Sets default values for missing fields in matching JSON responses.

Replace text using regex

Example Rule:

{ "match": { "url": "https://*.example.com/*", "content_type": "text/html" }, "transform": { "type": "text", "action": "regex", "pattern": "href=\"(https?://www\\.example\\.com/)", "replacement": "href=\"https://api.example.com/" } }

Applies regex pattern replacement in text responses.

Simple string replacement

Example Rule:

{ "match": { "url": "https://*.example.com/*", "content_type": "text/plain" }, "transform": { "type": "text", "action": "replace", "search": "www.example.com", "replace": "api.example.com" } }

Replaces exact text matches in text responses.
